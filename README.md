# Penyewaan Sarana Olahraga

Web Sistem Informasi untuk memudahkan penyewaan berbagai macam tempat / sarana olahraga.

## Features

-   Booking
-   Login / Register
-   Dashboard admin
-   Voucher
-   Cetak bukti Booking
-   CRUD Sarana,Voucher dll.
-   Laporan pdf

## Tech Stack

**Laravel, Bootstrap, Botstrap Admin Template, MySQL, Jquery**
